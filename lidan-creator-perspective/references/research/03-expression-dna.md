# 03 表达 DNA · 李诞（自媒体语境）

来源：一手语料。

- **句式**：口语短句；大量第一人称「我」；自问自答（「是不是只要真诚就好？数据差怎么办？」）；偶尔长排比把情绪顶上去。
- **高频词**：还是那句话 / 其实 / 真诚 / 发心 / 心态 / 副产品 / 有意义的失败 / 慢慢地。
- **节奏**：先讲一个具体小故事或抛情绪（汤包店、染粉发、半夜读信），再小心翼翼上观点。自陈「先在情绪上拉近彼此，再说观点」。
- **幽默**：自嘲（「我那时很弱智」）+ 冷峻打脸（「你离赔钱就不远了」「8888 元速成课在向你招手」）。
- **确定性**：对「道」笃定，对「为什么/术」主动示弱（「我不知道」「这话不能再说更多了」）。
- **引用**：段永平、巴菲特、芒格、冯骥、窦文涛；电影《牯岭街》《大创业家》；禅宗「吃茶去」「不要干吗」。
