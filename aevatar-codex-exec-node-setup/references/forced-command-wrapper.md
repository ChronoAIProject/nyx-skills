# Forced Command Wrapper

Install a wrapper that accepts only the exact command generated by Aevatar's `codex_exec`. Do not evaluate `SSH_ORIGINAL_COMMAND` with a shell.

Render the three absolute paths at the top, install the file with mode `0700`, and make it the SSH key or server `ForceCommand`.

```sh
#!/bin/sh
set -eu

WORKSPACE="/absolute/path/to/git-workspace"
CODEX_BIN="/absolute/path/to/codex"
CODEX_HOME_DIR="/absolute/path/to/codex-home"

prefix="p='"
suffix="'; { printf '%s' \"\$p\" | base64 --decode 2>/dev/null || printf '%s' \"\$p\" | base64 -D; } | codex exec -"

reject() {
    printf '%s\n' 'This SSH key only accepts Aevatar codex_exec requests.' >&2
    exit 126
}

original_command=${SSH_ORIGINAL_COMMAND:-}
case "$original_command" in
    "$prefix"*"$suffix") ;;
    *) reject ;;
esac

encoded_prompt=${original_command#"$prefix"}
encoded_prompt=${encoded_prompt%"$suffix"}
case "$encoded_prompt" in
    ''|*[!A-Za-z0-9+/=]*) reject ;;
esac

[ "${#encoded_prompt}" -le 8000 ] || reject
[ $(( ${#encoded_prompt} % 4 )) -eq 0 ] || reject

prompt_file=$(mktemp "${TMPDIR:-/tmp}/aevatar-codex-prompt.XXXXXX")
trap 'rm -f "$prompt_file"' EXIT HUP INT TERM
if printf '%s' "$encoded_prompt" | base64 --decode >"$prompt_file" 2>/dev/null; then
    :
elif printf '%s' "$encoded_prompt" | base64 -D >"$prompt_file" 2>/dev/null; then
    :
else
    reject
fi

prompt_bytes=$(wc -c <"$prompt_file" | tr -d ' ')
[ "$prompt_bytes" -gt 0 ] && [ "$prompt_bytes" -le 6000 ] || reject

export HOME=$(dirname "$CODEX_HOME_DIR")
export CODEX_HOME="$CODEX_HOME_DIR"
export PATH=$(dirname "$CODEX_BIN"):/usr/local/bin:/usr/bin:/bin:/usr/sbin:/sbin
export NO_COLOR=1

cd "$WORKSPACE"
exec 3<"$prompt_file"
rm -f "$prompt_file"
trap - EXIT HUP INT TERM
exec "$CODEX_BIN" exec \
    --ephemeral \
    --sandbox workspace-write \
    -c 'approval_policy="never"' \
    -C "$WORKSPACE" \
    - <&3
```

Requirements:

- `WORKSPACE` must be an existing git repository and must not be user-controlled per request.
- `CODEX_HOME_DIR` must already contain this principal's valid Codex authentication/configuration.
- The wrapper and its parent directory must not be writable by unrelated users.
- Do not add `--dangerously-bypass-approvals-and-sandbox`.
- Reject generic SSH commands with exit code `126` before testing Codex.
